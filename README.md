# FileMover.py
This simple script allows you to move thousands of files and sort them all in a single folder. 

## Requirements

- Python 3.8

## Some more info:
I am aiming on adding more and more formats in the future. Currently the only formats available from:

**Audio:**
 - .MP3 
 - .WAV
 - OGG
 - AAC
 - WMA
 - FLAC
 - M4A

**Images**:
 - .JPG 
 - .PNG
 -  .GIF 
 - .WEBP 
 - .TIFF 
 - .PSD
 - .RAW 
 - .BMP 
 - .HEIF 
 - .INDD 
 - .JPEG

**Word Processing**:
 - .TXT
 - .DOCX
 - .RTF
 - .PDF
